# Changelog

## 2.21.0

- Initial release with ntfy v2.21.0
- Support for all major architectures (amd64, aarch64, armv7, armhf, i386)
- Configurable HTTP port
- Optional attachment support
- Optional authentication and access control
- Timezone configuration
- Firebase Cloud Messaging support (optional)
- Email notifications support (optional)
- Web interface included
- Health check endpoint
